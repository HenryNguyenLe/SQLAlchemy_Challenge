Planning for vacation can be quite an intensive task. One of the most important factors to consider when choosing an ideal place is its weather condition. On this simple website, users can quickly query weather condition based on start date, or start and end dates.

All data can be accessed thru APIs below:

Quick APIs to all available data:

  • Precipitations
  • Weather Station List
  • Temperature Observations of the Most Active Station

To query data on a specific start date or start-end dates, use the following APIs

NOTE: all dates are in format YYYY-MM-DD

  • /api/startdate=<start-date>
  • /api/startdate=<start>/enddate=<end>/
